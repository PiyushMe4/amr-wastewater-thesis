# Data Management

This directory tracks metadata and accessions for publicly available datasets used in the study.

## Structure

- `metadata/`: CSV/TSV files containing sample details, geolocation, and accession numbers (NCBI/SRA).
- `processed/`: Intermediate processed results (e.g., ARG abundance tables).

_Note: Raw sequencing data (FASTQ) is not stored in this repository to maintain size limits._
