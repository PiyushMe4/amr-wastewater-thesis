# Project Documentation

This directory contains the core documentation for the research project: _"Medical Waste Influence on AMR Gene Patterns in Tier-2 Indian City Wastewater"_.

## Key Documents

- **[Implementation Log](IMPLEMENTATION_LOG.md)**: A continuously updated record of procedures, technical milestones, and project history.
- **Visual Synopsis**: Detailed descriptions of the figures located in the `visualization/` directory.

## Contents

- **Methodology Notes**: Documentation of the bioinformatics pipelines and statistical models.
- **Literature Summaries**: Key takeaways from relevant research papers.
- **Analysis Decisions**: Records of why specific parameters or datasets were chosen.
